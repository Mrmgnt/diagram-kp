title Sequence Diagram - Proses Registrasi (Pola MVC)
hide footbox

actor Pengguna
boundary "Halaman_Registrasi" as Halaman_Registrasi
control "Controller" as Controller
entity "Entity" as Entity

Pengguna -> Halaman_Registrasi: Mengakses halaman registrasi
activate Halaman_Registrasi
Halaman_Registrasi --> Pengguna: Menampilkan form registrasi
deactivate Halaman_Registrasi

Pengguna -> Halaman_Registrasi: Mengisi data form dan klik "Register"
activate Halaman_Registrasi

Halaman_Registrasi -> Controller: Meneruskan data registrasi
activate Controller

Controller -> Entity: Panggil fungsi validasi data(data_form)
activate Entity
Entity -> Entity: Melakukan validasi (format, duplikasi email/username)
Entity --> Controller: Mengembalikan hasil validasi (valid / tidak valid)
deactivate Entity

alt Data Valid
    Controller -> Entity: Panggil fungsi buatUserBaru(data_form)
    activate Entity
    Entity -> Entity: Menyimpan data pengguna baru ke database
    Entity --> Controller: Konfirmasi pengguna telah dibuat
    deactivate Entity
    
    Controller --> Halaman_Registrasi: Respon sukses, arahkan ke halaman login
    deactivate Controller
    
    Halaman_Registrasi --> Pengguna: Menampilkan halaman Login
    deactivate Halaman_Registrasi
else Data Tidak Valid
    Controller --> Halaman_Registrasi: Respon gagal dengan pesan error spesifik
    deactivate Controller
    
    Halaman_Registrasi --> Pengguna: Menampilkan pesan error di form registrasi
    deactivate Halaman_Registrasi
end

@enduml
