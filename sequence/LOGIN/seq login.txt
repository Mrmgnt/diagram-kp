!theme materia

title Sequence Diagram - Proses Login (Pola MVC)
hide footbox

actor Pengguna
boundary "View" as View
control "Controller" as Controller
entity "Model" as Model

Pengguna -> View: Mengakses halaman login
activate View
View --> Pengguna: Menampilkan halaman login
deactivate View

Pengguna -> View: Input username & password, lalu klik login
activate View

View -> Controller: Meneruskan permintaan login (username, password)
activate Controller

Controller -> Model: Panggil fungsi autentikasi(username, password)
activate Model

Model -> Model: Validasi kredensial pengguna\n(berinteraksi dengan database)

Model --> Controller: Mengembalikan hasil autentikasi (berhasil / gagal)
deactivate Model

alt Autentikasi Berhasil
    Controller --> View: Respon sukses, arahkan ke dashboard
    deactivate Controller
    
    View --> Pengguna: Menampilkan halaman Dashboard
    deactivate View
else Autentikasi Gagal
    Controller --> View: Respon gagal dengan pesan error
    deactivate Controller
    
    View --> Pengguna: Menampilkan pesan "Username atau password salah"
    deactivate View
end

@enduml
