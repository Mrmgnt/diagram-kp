!theme materia

title Sequence Diagram - Menampilkan Dashboard (Pola MVC)
hide footbox

actor "Pengguna Sistem" as Pengguna
boundary "Halaman_Dashboard" as Halaman_Dashboard
control "Controller" as Controller
entity "Entity" as Entity

Pengguna -> Halaman_Dashboard: Membuka menu/halaman "Dashboard"
activate Halaman_Dashboard

Halaman_Dashboard -> Controller: Minta data untuk dashboard
activate Controller

Controller -> Entity: Panggil fungsi getDashboardData()
activate Entity
Entity -> Entity: Mengambil & mengolah data dari berbagai sumber
Entity --> Controller: Mengembalikan data yang sudah diproses
deactivate Entity

Controller --> Halaman_Dashboard: Meneruskan data dashboard
deactivate Controller

Halaman_Dashboard --> Pengguna: Menampilkan halaman Dashboard dengan visualisasi data
deactivate Halaman_Dashboard

Pengguna -> Halaman_Dashboard: Berinteraksi dengan filter (contoh: rentang tanggal)
activate Halaman_Dashboard

Halaman_Dashboard -> Controller: Minta data dashboard dengan filter baru
activate Controller
' ... proses pengambilan data dengan filter baru akan berulang ...'
Controller --> Halaman_Dashboard: Meneruskan data yang sudah difilter
deactivate Controller

Halaman_Dashboard --> Pengguna: Memperbarui visualisasi data di dashboard
deactivate Halaman_Dashboard
