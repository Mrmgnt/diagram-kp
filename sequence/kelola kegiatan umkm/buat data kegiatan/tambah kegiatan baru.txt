@startuml
!theme materia

title Sequence Diagram - Tambah Data Kegiatan UMKM (Pola MVC)
hide footbox

actor "Pengelola" as Pengguna
boundary "View" as View
control "Controller" as Controller
entity "Model" as Model

Pengguna -> View: Memilih opsi "Tambah Kegiatan Baru"
activate View
View --> Pengguna: Menampilkan form tambah kegiatan
deactivate View

Pengguna -> View: Mengisi data form & klik "Simpan"
activate View

View -> Controller: Meneruskan data kegiatan baru
activate Controller

Controller -> Model: Panggil fungsi validasi(data_kegiatan)
activate Model
Model -> Model: Melakukan validasi data form
Model --> Controller: Mengembalikan hasil validasi (valid / tidak valid)
deactivate Model

alt Data Valid
    Controller -> Model: Panggil fungsi simpanDataKegiatan(data_kegiatan)
    activate Model
    Model -> Model: Menyimpan data kegiatan baru ke database
    Model --> Controller: Konfirmasi data telah disimpan
    deactivate Model
    
    Controller --> View: Respon sukses
    deactivate Controller
    
    View --> Pengguna: Menampilkan notifikasi "Data Kegiatan UMKM berhasil ditambahkan"
    deactivate View
else Data Tidak Valid
    Controller --> View: Respon gagal dengan pesan error validasi
    deactivate Controller
    
    View --> Pengguna: Menampilkan pesan error di form
    deactivate View
end

@enduml
