@startuml
!theme materia

title Sequence Diagram - Tambah Data Kegiatan UMKM (Pola MVC)
hide footbox

actor "Pengelola" as Pengguna
boundary "Halaman_kelola_KegiatanUMKM" as Halaman_kelola_KegiatanUMKM
control "Controller" as Controller
entity "Entity" as Entity

Pengguna -> Halaman_kelola_KegiatanUMKM: Memilih opsi "Tambah Kegiatan Baru"
activate Halaman_kelola_KegiatanUMKM
Halaman_kelola_KegiatanUMKM --> Pengguna: Menampilkan form tambah kegiatan
deactivate Halaman_kelola_KegiatanUMKM

Pengguna -> Halaman_kelola_KegiatanUMKM: Mengisi data form & klik "Simpan"
activate Halaman_kelola_KegiatanUMKM

Halaman_kelola_KegiatanUMKM -> Controller: Meneruskan data kegiatan baru
activate Controller

Controller -> Entity: Panggil fungsi validasi(data_kegiatan)
activate Entity
Entity -> Entity: Melakukan validasi data form
Entity --> Controller: Mengembalikan hasil validasi (valid / tidak valid)
deactivate Entity

alt Data Valid
    Controller -> Entity: Panggil fungsi simpanDataKegiatan(data_kegiatan)
    activate Entity
    Entity -> Entity: Menyimpan data kegiatan baru ke database
    Entity --> Controller: Konfirmasi data telah disimpan
    deactivate Entity
    
    Controller --> Halaman_kelola_KegiatanUMKM: Respon sukses
    deactivate Controller
    
    Halaman_kelola_KegiatanUMKM --> Pengguna: Menampilkan notifikasi "Data Kegiatan UMKM berhasil ditambahkan"
    deactivate Halaman_kelola_KegiatanUMKM
else Data Tidak Valid
    Controller --> Halaman_kelola_KegiatanUMKM: Respon gagal dengan pesan error validasi
    deactivate Controller
    
    Halaman_kelola_KegiatanUMKM --> Pengguna: Menampilkan pesan error di form
    deactivate Halaman_kelola_KegiatanUMKM
end

@enduml
